==========================================
  SESpec Docker 离线打包工具
==========================================

使用方法:
  1. 在有网络的机器上运行: ./pack-offline.sh
  2. 这将生成 sespec-offline.tar
  3. 拷贝 sespec-offline.tar 到离线机器
  4. 在离线机器上运行: docker load -i sespec-offline.tar
  5. 运行: docker run -it sespec:offline bash
