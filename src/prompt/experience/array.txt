Examples:
Use the following array-oriented invariant patterns as prompt engineering guidance.

- Capture index bounds explicitly.
- State which array segments are unchanged.
- State relational properties over processed prefixes or suffixes with `\forall`.
- If the loop accumulates a summary, relate the summary variable to the processed array region.

Example pattern:
```c
/*@
  loop invariant 0 <= i <= n;
  loop invariant \forall integer k; 0 <= k < i ==> a[k] <= max;
  loop invariant \forall integer k; i <= k < n ==> a[k] == \at(a[k],Pre);
  loop assigns i, max;
*/
```

For arrays, prefer explicit segment properties over vague summaries.
