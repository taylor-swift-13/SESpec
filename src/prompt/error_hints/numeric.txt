### Category-specific pitfalls — Numeric (scalar / control-flow only, no pointers) ###

B1. ACSL has NO `\max`, NO `\min`, NO `\abs` builtins. Expand with conditionals (`a > b ? a : b`, `x < 0 ? -x : x`) or define a `logic` function inside `/*@ ... */` (e.g. `logic integer my_abs(integer x) = x < 0 ? -x : x;`) and use that. Numeric loops over absolute values / max-of-N hit this most often.

B2. Loop labels: ACSL has `Pre`, `Old`, `Post`, `Here`, `LoopCurrent`. There is NO `LoopEntry`. Use `\at(var, Pre)` or rely on the value already bound in the precondition.
